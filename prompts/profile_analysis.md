# Промпт для анализа базы блогеров

Этот промпт нужен для этапа, где модель получает список уже отобранных блогеров и должна собрать черновой портрет подходящего инфлюенсера для бренда женской одежды.

## Цель

- выделить общие темы контента;
- выделить стиль подачи;
- выделить визуальные паттерны;
- описать, какие аккаунты стоит искать дальше.

## Шаблон промпта

```text
Ты анализируешь базу блогеров для бренда женской одежды, который продает товары на маркетплейсах.

Ниже дан список блогеров и признаки по ним:
- username
- платформа
- темы
- визуальные теги
- tone of voice
- дополнительные заметки

Твоя задача:
1. Выдели общие паттерны, которые повторяются у большинства блогеров.
2. Сгруппируй выводы по блокам:
   - темы контента
   - стиль подачи
   - визуальная эстетика
   - коммерческая пригодность для бартерного сотрудничества
3. На основе этого составь портрет "идеального блогера".
4. Отдельно перечисли признаки, по которым можно искать новых похожих блогеров.

Формат ответа:
- Краткая сводка
- Темы контента
- Стиль подачи
- Визуальная эстетика
- Признаки для поиска новых кандидатов
- Риски и ограничения анализа
```

## Как использовать в нашем MVP

В этот промпт можно подставлять:

- данные из `outputs/source_bloggers_features.json`
- агрегированный профиль из `outputs/ideal_blogger_profile.json`
- при необходимости расширенные данные по bio или описаниям контента

## Комментарий

Сейчас в проекте портрет базы в основном строится эвристически кодом, но этот промпт можно использовать как следующий слой интерпретации и упаковки анализа в человеческий вид.
