# Промпт для генерации персональных офферов

Этот промпт нужен для этапа, где уже выбраны новые кандидаты, и нужно сгенерировать для каждого теплое персонализированное предложение о бартерном сотрудничестве.

## Цель

- написать короткое и нативное сообщение;
- объяснить, почему бренд пишет именно этому блогеру;
- избежать шаблонного рекламного тона.

## Шаблон промпта

```text
Ты помогаешь бренду женской одежды подготовить персональное сообщение блогеру для бартерного сотрудничества.

Данные о бренде:
- категория: женская одежда
- формат сотрудничества: бартер
- цель: предложить вещи на обзор или интеграцию

Данные о блогере:
- username: {username}
- платформа: {platform}
- темы контента: {matched_topics}
- визуальные теги: {matched_visual_tags}
- tone of voice: {matched_tone}
- причины выбора: {reasons}

Твоя задача:
1. Напиши короткое персональное сообщение.
2. Упомяни, что именно понравилось в контенте блогера.
3. Сделай тон теплым, уважительным и нативным.
4. Не делай сообщение слишком длинным.
5. Не используй агрессивные продажи.

Формат ответа:
- Тема сообщения
- Текст сообщения
- Короткое пояснение, какие элементы были использованы для персонализации
```

## Как использовать в нашем MVP

В этот промпт можно подставлять:

- данные из `outputs/top_candidate_matches.json`
- список `matched_topics`
- список `matched_visual_tags`
- список `matched_tone`
- `reasons` из матчинга

## Комментарий

Сейчас в проекте офферы собираются кодом по шаблону, но этот промпт показывает, как тот же этап может быть усилен через LLM при наличии доступного API.
